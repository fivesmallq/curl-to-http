curl-to-java

Instantly convert curl commands to Java code

This tool turns a curl command into Java code. Currently, it knows the following options: -d/--data, -H/--header, -I/--head, -u/--user, --url, and -X/--request. Feel free to contribute on GitHub!

This script derives from Matt Holt's excellent curl-to-Go and curl-to-php.
